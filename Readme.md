# Findmyva Test for Tony Tripulca

Create a simple script that will scrape of fetch race details from [sportsbet.com.au](https://sportsbet.com.au)

## Setup involved in the VM

1. Installed nodejs
2. Installed git
3. Installed vs code

## Run locally

### Dependency

1. Nodejs 22.11.0
2. Axios 1.7.7
3. Moment 2.30.1

### Execute

1. Clone this repo
2. Go to repo dir
3. Install packages `npm install`
4. Run `npm start`
